# Pomodoro Timer

Sci-fi themed pomodoro timer. Single HTML file, no dependencies.

**Live**: https://aretedriver.github.io/pomodoro-timer/

## Features

- **Timer modes**: 25min Pomodoro, 15min Focus, 50min Deep Work, 5/15min breaks
- **Stats tracking**: Completed sessions, total minutes, streak (persisted to localStorage)
- **Browser notifications**: Alerts when timer completes (works in background)
- **Audio alerts**: Tone plays on session completion
- **Keyboard shortcuts**: `Space` to start/pause, `R` to reset
